AGEX
====
Sistema de Agendamento de Exames

## Introdução #
O _AGEX_ é um Sistema de Agendamento de Exames open-source desenvolvido em _Java_. Ele é desenvolvido utilizando o framework Java Server Faces. E sua interface já é HTML juntamente com bootstrap.

## Características #
O _AGEX_ é uma aplicação WEB desenvolvida com o objetivo de facilitar o agendamento de exames de pacientes de um determinado hospital. Ele também permite o usuário cadastrar e consultar médicos, exames, e organizar todos esses dados em uma agenda.
