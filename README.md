AGEX
====

## Introdução #
O _AGEX_ é um Sistema de Agendamento de Exames open-source desenvolvido em _Java_.

## Características #
O _AGEX_ O _AGEX_ é uma aplicação WEB desenvolvida com o objetivo de facilitar o agendamento de exames de pacientes de um determinado hospital. Ele também permite o usuário cadastrar e consultar médicos, exames, e organizar isso tudo em uma agenda.
